# PetsHosts


## Pre-Requisites

- Ruby 2+
- RubyGems
- Git

## Setup

1. First, we need to install the project dependencies:

    ```
    bundle install
    ```

2. Then, we need to run the database migrations:

    ```
    rake db:migrate
    ```

3. Finally, you can run the server:

    ```
    rails server
    # Or just
    rails s
    ```
